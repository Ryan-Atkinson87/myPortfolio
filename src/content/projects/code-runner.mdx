Autonomous AI coding agents are powerful but unstructured out of the box: they forget context between sessions, stray beyond their remit, make calls they shouldn’t, and have no idea when they’ve hit their usage ceiling. Running a multi-repo project through one is manual and fragile.

Code Runner wraps the agent in a proper orchestration layer. It drives Claude Code (or Codex or Gemini, behind a common adapter) through a project wave by wave — managing branches, PRs, Notion sync, GitHub issue state and usage monitoring automatically. The human stays out until a wave hand-off PR is ready for review. When capacity runs low, the engine pauses rather than degrading quality silently.

The first target is Trive Services, a real three-repo platform. Code Runner is built and validated against it, then extracted as a generic tool that runs on any project with minimal setup.
