I'm Ryan, a software engineer based in Burnley. Most of what I build comes from running into a problem myself and deciding to fix it properly rather than put up with it.

I keep tropical fish, so I built an open-source monitor for them. I ride indoors on Zwift, so I built a workout editor the existing tools didn't offer. My son asked for a robot we could actually control, so we built Robert together. Each one started the same way: a real problem, built out properly, then taken far enough that other people can use it too.

These projects ship under Trive Dev, my own venture. It's where I own the whole thing: the architecture, the design system, the infrastructure, and the AI workflows I use to work faster.

I try to finish things properly, with authentication, accessibility, error tracking and real deployments, not just a working demo. I'm looking for engineering work where that end-to-end ownership is useful, either in a team or with clients who need someone who can take a project from idea to live.
